Marcos no sabe usar git ni GitHub. Para cualquier tarea que implique repositorios, push, GitHub Actions o despliegue, hay que dar instrucciones paso a paso, concretas y sin asumir conocimientos previos (dónde hacer clic, qué comando exacto copiar).

Prefiere soluciones que no dependan de tener su PC encendido (p. ej. cron en la nube vía GitHub Actions).
